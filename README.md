# ✊📄✂️ Rock Paper Scissors Game

## 🌟 Overview

An interactive Rock Paper Scissors game developed using **HTML5, CSS3, and JavaScript**. Players compete against the computer in a 15-round challenge, with real-time score tracking, move counting, result evaluation, and a dedicated results screen displaying the final winner.

The project demonstrates core front-end development concepts including DOM manipulation, event handling, game logic implementation, and responsive user interface design.

---

## 🎮 Features

### 🚀 Multi-Page Game Flow

* Welcome screen with game start option
* Interactive gameplay screen
* Dedicated results page with final score summary
* Option to restart and play again

### ✊ Rock Paper Scissors Gameplay

* Choose between Rock, Paper, or Scissors
* Computer generates random moves
* Instant round-by-round outcome evaluation

### 📊 Score Tracking

* Real-time player score updates
* Real-time computer score updates
* Dynamic scoreboard display

### 🔢 Move Counter

* Fixed 15-round game format
* Tracks total moves played
* Automatically proceeds to final result after all rounds

### 🏆 Winner Announcement

* Displays final winner
* Shows complete score summary
* Celebration screen for game completion

### 🎨 Interactive UI

* Image-based game controls
* Clean and engaging layout
* Responsive and user-friendly interface

* Scissors beats Paper

* Each match consists of 15 rounds
* The player with the highest score at the end wins
